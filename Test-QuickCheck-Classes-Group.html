Group

groupLaws :: (Arbitrary a, Show a, Eq a, Group a) => Proxy a -> Laws Source #

Laws for instances of Group.

Includes the following laws:

Inversion

invert mempty == mempty

invert (invert a) == a

       a <> invert a == mempty
invert a <>        a == mempty

Subtraction

a ~~ mempty == a

a ~~ a == mempty

a ~~ b == a <> invert b

Exponentiation

pow a 0 == mempty

n >= 0 ==> pow a n ==         mconcat (replicate      n  a)
n <= 0 ==> pow a n == invert (mconcat (replicate (abs n) a))
